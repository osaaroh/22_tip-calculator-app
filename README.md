### What I learned

- CSS :focus-visible and :disabled pseudo-elements

```css
.btn-reset:hover,
.btn-reset:active,
.btn-reset:focus-visible {
    cursor: pointer;
    background: var(--light-grayish-cyan);
    color: var(--very-dark-cyan);
}

.btn-reset:disabled {
    background-color: var(--dark-grayish-cyan);
    opacity: 0.7;
}

.btn-reset:disabled:hover {
    background-color: var(--dark-grayish-cyan);
    cursor: unset;
}

```
